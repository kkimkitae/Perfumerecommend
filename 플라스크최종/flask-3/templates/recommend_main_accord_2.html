감귤, 오렌지, 레몬, 유자, 라임, 자몽 등의 과일의 향

citrus : 과일 내음과 약간은 꽃내음이 나며 신선하고 상큼하면서 동시에 달콤한 느낌의 향




시트러스를 제외한 과일, 야채, 견과류를 포함하는 향

fruity : 다양한 열대과일을 이용하여 상큼하고 달콤한 과일 향
coconut : 달콤함이 느껴지는 진한 코코넛 향
cherry : 달콤하면서도 시큼한 동시에 특징적인 향
tropical : 시원하고 달콤한 열대 과일 주스처럼 활력을 넣어주는 트로피컬 향
almond : 씁쓸하면서도 달콤한 향
nutty : 볶은 견과류 향
sour : 시큼한 향
conifer : 상쾌한 향




장미, 재스민, 라일락, 카네이션,튜버로즈,히아신스,네롤리,바이올렛, 수선화 향

iris : 바이올렛 플라워를 연상시키는 파운더리한 플로럴 향
floral : 꽃의 향수, 실급 타입부터 부케 타입까지 수수함과 화려한 향
lavender : 포근하고 아늑한 느낌의 향
rose : 은은하면서 향기로운 장미향
violet : 스윗함과 파우더리함을 지닌 향
tuberose : 은은하면서 차분한 향
aldehydic : 신선한 바람과 같은 향




자스민, 가드니아, 튜베로즈를 연상시키는 묵직한 꽃 향

white floral : 새하얗고 두터운 꽃잎을 꼭 닮은 벨벳과 같이 풍부하고 크리미한 향
yellow floral : 전체적으로 부드러운 향




풀과 허브, 담뱃잎의 향

tobacco : 건초의 향과 함께 풍부하고 뉘앙스의 따뜻하고 달콤한 허브 향
green : 막 베어낸 풀이나, 나뭇잎을 비볐을 때의 향
aromatic : 신선하고 깔끔한 느낌의 향
cannabis : 더러운 시큼한 뉘앙스가 있는 향
bitter : 씁슬하고 쓴 향




계피, 후추, 정향, 고수풀, 생강, 커피같은 천연 향신료의 향

warm spicy : 클로브,시나몬,페퍼와 같은 자극적인 향
vanilla : 달달하며 부드럽고 은은한 향
fresh spicy : 생허브잎같이 쏘는 듯한 향
cinnamon : 달콤하고 따뜻하며 가루 같은 끈끈한 매운향
coffe : 커피의 맛있는 향
cacao : 앰버, 발사믹 뉘앙스가 있는 따뜻하고 스파이시한 향
anis : 감초를 연상시키는 아네톨이 풍부한 향
oily : 기름 향




초콜릿, 생크림, 캐러멜 향부터 마카롱, 크렘 브륄레같은 달달한 디저트 향

sweet : 폭신하고 편안한 향
chocolate : 진하고 신선하면서 달콤한 향
honey : 그윽하고 깊은 향을 가진 허니 향
caramel : 관능적이며 풍부하고 크리미한 버터 같은 향




숲은 연상시키는 나무와 이끼 계열 향

parchouli : 나무 냄새, 흙 냄새 비슷한 우디 향
oud : 나무 향뿐만 아니라 열매와 고소한 향




나무 진액같이 묵직하고 진한 향

balsamic : 풍성하고 복합적인 발삼 노트 향
rubber : 은은하고 묵직한 고무 향




따뜻한, 파우더리한, 에로틱한, 약간의 애니멀릭한 향

mossy : 축축한 이끼, 살짝 젖어있는 대지 등을 연상하는 향
animalic : 관능적이고 섹시한 느낌을 주는 향
lactonic : 빡빡하고 깨끗한 느낌 향
beeswax : 강렬한 허니 뉘앙스가 있는 깊고 향긋한 앰버 향




콜라, 칵테일 같은 음료의 향

coca-cola : 계피, 라임, 레몬, 오렌지, 고수풀, 바닐라, 육두구가 결합된 실제 청량 음료 같은 향

Champagne : 톡쏘고 상큼한 향
vodka : 무색 무취의 향
alcohol : 술의 주 성분이며 무채색 향
